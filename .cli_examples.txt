# CLI Examples (copy/paste)

Notes
- Name form (e.g., `debates.yaml`) resolves the packaged preset. Use a file path (e.g., `src/agentrylab/presets/debates.yaml`) if you want to run a locally edited copy.
- Quote messages with single quotes to avoid shell history expansion.
- Ensure providers are available (e.g., `ollama serve`, `OPENAI_API_KEY`, `WOLFRAM_APP_ID`).


# Solo Chat With User — interactive chat (llama3-friendly)
  src/agentrylab/presets/solo_chat_user.yaml
  # Reset transcript for a clean start
  agentrylab reset solo_chat_user.yaml demoit --delete-transcript
  # Run interactively as user (user turn enabled)
  agentrylab run solo_chat_user.yaml --thread-id demoit --resume --max-iters 5 --interactive --user-id user --objective ""
  # Show status after run
  agentrylab status solo_chat_user.yaml demoit
  # List all threads for this preset
  agentrylab ls solo_chat_user.yaml
  # Reset again to clean up
  agentrylab reset solo_chat_user.yaml demoit --delete-transcript

# Solo Chat — non-interactive with say and extend (llama3-friendly)
  src/agentrylab/presets/solo_chat_user.yaml
  # Reset transcript for a clean start
  agentrylab reset solo_chat_user.yaml demo2 --delete-transcript
  # Queue a user message (non-interactive)
  agentrylab say solo_chat_user.yaml demo2 'Hello, agent!' --user-id user
  # Run one tick to process the user message
  agentrylab run solo_chat_user.yaml --thread-id demo2 --resume --max-iters 1 --show-last 5
  # Extend the run by 2 more turns
  agentrylab extend solo_chat_user.yaml demo2 --add-iters 2
  # Show status after extension
  agentrylab status solo_chat_user.yaml demo2
  # List all threads for this preset
  agentrylab ls solo_chat_user.yaml
  # Reset again to clean up
  agentrylab reset solo_chat_user.yaml demo2 --delete-transcript


# Debates — pretty run (+ objective override) (better with gpt-4o-mini)
  src/agentrylab/presets/debates.yaml
  # Reset transcript for a new debate
  agentrylab reset debates.yaml debate1 --delete-transcript
  # Run 4 turns with a custom objective
  agentrylab run debates.yaml --thread-id debate1 --objective "Proposition: apples — good or scam?" --max-iters 4 --show-last 10
  # Extend the debate by 2 more turns
  agentrylab extend debates.yaml debate1 --add-iters 2
  # Show status for this thread
  agentrylab status debates.yaml debate1
  # List all debate threads
  agentrylab ls debates.yaml
  # Reset again to clean up
  agentrylab reset debates.yaml debate1 --delete-transcript
